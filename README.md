# LC-3 Virtual Machine 

A simple lc-3 vm made by following the [blog](www.jmeiners.com/lc3-vm/) by Justin Meiners and Ryan Pendleton.

## Getting Started

### Dependencies

Electricity and an OS installed (preferrably Linux idk about Windows)

### Executing program

* How to run the program
  You'll have to pass the instructions in binary format. It takes a 16 bit number as instruction.
```
lc-3 <instruction and parameters>
```
